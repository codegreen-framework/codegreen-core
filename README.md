The core business logic of codegreen



# Installation 

- Install the package
- Create a config file : `.codegreencore.config`
- Initial config file:
  ```
  [codegreen]
  ENTSOE_token = 1234
  CO2_SIGNAL_TOKEN =  1234
  ```
  - To get ENTSOE token from the [transparency platform](https://transparency.entsoe.eu/dashboard/show)
